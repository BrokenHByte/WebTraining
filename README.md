# Учебный проект
Проект для практики навыков по созданию сервисов в фреймворке ASP.NET Core

# Сборка и запуск
1. Клонировать репозиторий с необходимой веткой
2. Запустить из папки проекта команду dotnet run
3. Можно тестировать по адресу
`http://localhost:5008/swagger/index.html`

# Доступное API
### Получить все события
GET /events  
**Пример ответа:**
```json
[
  {
    "Id": 1,
    "Title": "Концерт",
    "Description": "Описание",
    "StartAt": "2026-04-27T03:11:08.393Z",
    "EndAt": "2026-04-27T06:11:08.393Z"
  },
  {
    "id": 2,
    "title": "Заголовок2",
    "description": "Описание2",
    "startAt": "2026-06-27T03:11:08.393Z",
    "endAt": "2026-06-27T06:11:08.393Z"
  }
]
```
------------------------
### Добавить событие
POST /events  
**Тело запроса**  
```json
{
"title": "Заголовок",
"description": "Описание",
"startAt": "2026-04-27T03:11:08.393Z",
"endAt": "2026-04-27T06:11:08.393Z"
}
```
**Возвращает**  
201 при успехе  
------------------------
### Получить событие по значению Id
GET /events/{id}  
**Пример ответа:**
```json
{
  "id": 1,
  "title": "Заголовок",
  "description": "Описание",
  "startAt": "2026-04-27T03:11:08.393Z",
  "endAt": "2026-04-27T06:11:08.393Z"
}
```
**Возвращает**  
200 при успехе  
404 если событие не найдено
------------------------
### Изменить событие по Id
PUT /events/{id}  
**Тело запроса**
```json
{
  "title": "Новый заголовок",
  "description": "Описание",
  "startAt": "2026-04-27T03:11:08.393Z",
  "endAt": "2026-04-27T06:11:08.393Z"
}
```
**Возвращает**  
200 при успехе  
404 если событие не найдено  
400 если есть нарушения в теле запроса
------------------------
### Удалить событие по Id
DELETE /events/{id}  

**Возвращает**  
200 при успехе  
404 если событие не найдено
